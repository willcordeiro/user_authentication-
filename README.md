### Installing

**Cloning the Repository**

```
$ git clone https://github.com/willcordeiro/user_authentication-

$ cd ./<project page>
```

**Installing dependencies**

```
$ yarn
```

_or_

```
$ npm install
```

**Starting the project**

```
$ node index.js
```

the project will run at the default port setted by me http://localhost:5000;/
